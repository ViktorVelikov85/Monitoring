# Ръководство за инсталация и стартиране

### Системни изисквания:
* .NET SDK 6.0 или по-нов.
* XAMPP Control Panel (с пуснати Apache и MySQL).
* Visual Studio 2022.

### Стъпки за пускане:
1. Отворете XAMPP и стартирайте модулите **Apache** и **MySQL**.
2. Влезте в `http://localhost/phpmyadmin/`.
3. Създайте нова празна база данни с име `network_monitoring`.
4. Импортирайте предоставения файл `network_monitoring.sql`.
5. Отворете сорс кода във Visual Studio.
6. Уверете се, че пакетът `MySql.Data` е инсталиран от NuGet Package Manager.
7. Стартирайте приложението (клавиш F5).
